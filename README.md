# Website Portfolio Pribadi

Halo! Saya Dea Ayu Novita Putri,

Proyek ini adalah Website Portfolio Pribadi

Website Portfolio Dea Ayu Novita Putri adalah website portofolio pribadi yang dibangun menggunakan HTML, CSS, dan JavaScript tanpa framework tambahan.
Website ini terdiri dari 6 section yaitu Home, About, Skills, Projects, Experience, dan Contact, semuanya dalam satu halaman yang bisa diakses lewat navbar di atas.
Untuk fiturnya, navbar akan otomatis highlight menu yang sedang aktif sesuai posisi scroll. Di tampilan HP, menu berubah jadi tombol hamburger yang bisa dibuka tutup. Setiap elemen seperti card skill, project, dan timeline muncul dengan animasi halus dari bawah saat discroll. Di bagian Projects, tombol Live Demo dan GitHub bisa langsung diklik dan akan membuka halaman yang dituju di tab baru. Begitu juga tombol GitHub dan Instagram di section Contact, langsung terhubung ke profil yang sesuai. Seluruh tampilan juga responsif, menyesuaikan diri baik di layar laptop maupun HP.
